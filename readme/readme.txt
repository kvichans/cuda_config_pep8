Plugin for CudaText.
It is helper for "Linter for Python using PEP8". It configurates PEP8 with dialog, and saves config file. File path is:

- Windows: c:\Users\name_here\.pep8
- Unixes: ~/.config/pep8 
          (or value of var XDG_CONFIG_HOME is used, if found)
